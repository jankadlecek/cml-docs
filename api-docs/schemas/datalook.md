# Datalook Schemas

> **Context for AI:** Datalook handles initialization and template management for data source systems.
> It provides templates for configuring connections to various source systems (e.g., Power BI)
> and manages transformation engine configurations for those sources.
>
> **See also:** [Datalook endpoints](../endpoints/datalook.md)

### `ICommandVariableTemplate`

Defines dynamic variables to be used for run. The key is name of the variable and value is variable template item.
Example:
```
{
    "SOURCE_DATABASE": {
        "value": null,
        "pathToConfiguration": "source.database"
     },
     "SOURCE_SCHEMA": {
         "value": null,
         "pathToConfiguration": ["source", "schema"]
     }
}
```

```json
{
  "description": "Defines dynamic variables to be used for run. The key is name of the variable and value is variable template item.\nExample:\n```\n{\n    \"SOURCE_DATABASE\": {\n        \"value\": null,\n        \"pathToConfiguration\": \"source.database\"\n     },\n     \"SOURCE_SCHEMA\": {\n         \"value\": null,\n         \"pathToConfiguration\": [\"source\", \"schema\"]\n     }\n}\n```",
```
